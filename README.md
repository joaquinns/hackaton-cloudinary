# Getting Started with the app

## Available Scripts

In the project directory, you can run:

### `pnpm run dev`

```
pnpm run dev
```

Runs the app in the development mode.


### `pnpm run build`

Start the script for build the app from typescript to javascript

```
pnpm run build
```
